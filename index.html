{ Close-Shadow-JS }

JavaScript generated 3D 'close' text shadows.

~

It makes flat text look positively dull.

Basic Usage

	$('h3').closeShadow();

It doesn't have to move

	$('#main h3#noMove').closeShadow({
		'follow': false
	});

Make the shadow a garish color

	$('#garish').closeShadow({
		'dark-shadow': 'pink',
		'light-shadow': '#00f2ff'
	});
